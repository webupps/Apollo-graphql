# The Art of Software in Action - GraphQL
Free Demo project for GraphQL

## Requirements 
Node Version
 - 20 
Install Nodemon: ``` npm install -g nodemon ```

### Run from the folder:
``` nodemon index.js ```
- visit  http://localhost:4000


### Run from the folder:
``` 
query ExampleQuery {
  books {
    title
  }
}
 ```
